Principais Formatação

Negrito / Destaque

Nesta frase temos um termo em Negrito usando a tag "b" (não semântico)

Nesta frase temos um termo em Negrito usando a tag "strong" (semântico)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag "i" (não semântico)

Nesta frase temos um termo em ênfase usando a tag "em" (semântico)

Podemos criar também um texto marcado usando a tag "mark"

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando as tags "big" e "small"

Textos deletador

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag "del"

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag "ins"

Podemos usar uma outra tag para marcar um texto como inserido usando a tag "u"

Textos sobrescritos

Para enserir coisas do tipo: x2 + y2 = (x + y)2 - 2xy
